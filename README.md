# 🌱 밭 볼텼유? — 신뢰 기반 농지 매칭 플랫폼  

> **데이터로 가능성을, 신뢰로 계약을**  
> 2025 멋쟁이사자처럼 해커톤 프로젝트  

---

## 📌 프로젝트 개요

**밭 볼텼유?**는 고령화와 정보 비대칭으로 인해 발생하는 **유휴농지 문제**를 해결하기 위한  
**신뢰 기반 농지 매칭 플랫폼**입니다.

농지를 보유한 어르신과 농지를 필요로 하는 청년을 단순히 연결하는 것이 아니라,  
청년의 활동 이력, 자격, 교육, 장비 보유 여부 등의 데이터를 기반으로 **신뢰도와 적합도**를 분석하여  
더 안전하고 설득력 있는 농지 임대·양도 매칭을 지원합니다.

---

## 💡 문제 정의 및 배경

### 해커톤 주제 재해석

- **해커톤 공식 주제:** *AI로 학교 주변 상권 살리기*
- 주제를 분석하는 과정에서, 학교 주변 지역의 특성상 **농지와 농업 기반 자원이 많다**는 점에 주목했습니다.
- 이에 상권 활성화를 단순 소비 증가 관점이 아니라, **지역 인구 유입과 1차 산업 기반 유지**의 관점에서 재해석했습니다.

### 왜 농지 매칭인가?

문제 인식은 가족 경험에서 출발했습니다.

> “평생을 바친 염전을 아무에게나 넘겨주고 싶지 않다.”

이 경험을 통해 농지 보유자에게 중요한 것은 단순한 거래 조건이 아니라,  
**상대방을 신뢰할 수 있는가**라는 점이라고 판단했습니다.

따라서 기존 해커톤 주제를 다음과 같이 AI로 해결 가능한 문제로 재정의했습니다.

- 농지 보유자는 **신뢰 부족과 정보 비대칭**으로 인해 임대·양도에 소극적임
- 청년 농업인은 농지 정보를 찾기 어렵고, 보유자에게 신뢰를 증명하기 어려움
- 농업 인력 고령화로 인해 농지 활용률 저하와 생산 기반 단절 위험이 커지고 있음
- 청년 농업인의 유입은 장기적으로 지역 인구 유입과 주변 상권 활성화로 이어질 수 있음

## 🎯 해결 방향

밭 볼텼유?는 다음 세 가지 방향으로 문제를 해결합니다.

1. **신뢰도 기반 매칭**
   - 청년의 자격증, 교육 이수, 장비 보유, 재배 이력, 추천인 정보를 기반으로 신뢰 점수를 산출합니다.

2. **AI 기반 적합도 분석**
   - 농지 조건과 청년의 희망 조건을 비교하여 적합한 매칭 후보를 추천합니다.

3. **사용자 맞춤형 UI/UX**
   - 청년에게는 상세한 농지 탐색 경험을 제공하고, 어르신에게는 큰 버튼과 단계형 입력 UI를 제공하여 접근성을 높였습니다.

---

## ✨ 주요 기능

### 1. 신뢰도 분석

청년 사용자의 정보를 기반으로 신뢰 점수를 산출합니다.

- 자격증 보유 여부
- 농업 교육 이수 여부
- 장비 보유 여부
- 재배 또는 임대 이력
- 추천인 정보

이를 통해 농지 보유자가 청년 사용자의 신뢰도를 직관적으로 판단할 수 있도록 설계했습니다.

---

### 2. AI 기반 농지 매칭

구매자·임차 희망자와 농지 보유자 간의 조건을 분석하여 적합도를 계산합니다.

- KMeans Clustering 기반 사용자/농지 유형 분류
- Rule-based Scoring 기반 조건 적합도 계산
- 예상 수익 예측 모델 제공
- 농지 조건과 청년 희망 조건 간 매칭 결과 제공

---

### 3. 사용자 맞춤 UI/UX

사용자 유형에 따라 다른 사용 경험을 제공합니다.

- **청년 사용자**
  - 지도 기반 농지 탐색
  - 상세 데이터 기반 필터링
  - 농지 정보 및 예상 수익 확인

- **어르신 사용자**
  - 큰 버튼 중심의 인터페이스
  - 단계별 농지 등록 흐름
  - 복잡한 입력을 줄인 간단한 사용 구조

---

### 4. 실시간 채팅

상호 동의가 이루어진 사용자 간에만 채팅이 열리도록 설계했습니다.

- 매칭 이후 채팅 가능
- 무분별한 연락 방지
- 안전한 거래 전 커뮤니케이션 지원

## 📌 프로젝트 의의

이 프로젝트는 단순히 농지를 거래하는 플랫폼이 아니라,
**지역 문제를 데이터와 신뢰 구조로 해결하려는 시도**입니다.

특히 해커톤 주제였던 “AI로 학교 주변 상권 살리기”를 직접적인 소비 서비스로 해석하지 않고,
지역의 농업 자원, 고령화, 청년 유입, 유휴농지 문제를 연결하여
**1차 산업 기반 유지와 지역 경제 활성화**라는 방향으로 확장했습니다.

이를 통해 밭 볼텼유?는 다음과 같은 가치를 목표로 합니다.

* 유휴농지 활용률 증가
* 청년 농업인의 진입 장벽 완화
* 농지 보유자의 신뢰 기반 의사결정 지원
* 지역 인구 유입 가능성 확대
* 장기적인 지역 상권 활성화 기여
